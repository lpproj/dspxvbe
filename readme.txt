[内容説明]

むかし作った IBM DOS/V Extension 用ビデオ拡張モジュールおよび付属ツールの
ソースその他を発掘したので、歴史的遺物として現状のまま公開してみます。
以下のモジュール用ソースが収められているはずです。

・VESA VBE 汎用
・Cirrus Logic CL-GD5426/5428/5434 用

WC90C24/31 用と VGA/SVGA 汎用のやつは見つからなかった…。

ほんとに発掘してきただけなので、現状のソースできちんと作れるどうか未確認です。
内容ならびに利用結果についてはいかなる保証ならびに補償もございません。
あしからず。

まあ、なんかの足しにってことで。


[構築方法]

当時使っていたツールは、

・Borland TASM 3.1, tlink, make（Borland C++ 3.1 付属のもの）
・Watcom wlink（Watcom C/C++ 10.5j 付属のもの）
・マイクロソフトの link（たしか MASM 5.1 か OS/2 Warp に付属していたやつ）

バージョン 6.0 以上の MASM（ml.exe）があれば tasm なしでもいけるかも。
